# Capacitor Basic With HTML - JS
## Install And Run
- Clone source
```
git clone https://github.com/ron19102004/capacitor-core-html-js.git
```
- Install package
```
npm install
```
- Sync code
```
npx cap sync
```
- Run app
```
npx cap open android
```
